# AgenRACI v0.1.2

A metadata-only release — **no functional change** from 0.1.1. The CLI, checker
rules (R1–R6), and every command behave identically.

## What changed

- The GitHub Action's name and description now lead with the word **RACI**
  (`AgenRACI — RACI charter checker`), so a Marketplace search for "raci" can
  find the listing. Nothing about how the Action runs changed.

If you're already on 0.1.1 there's nothing to upgrade for — this exists so the
Marketplace listing carries the searchable metadata.

```bash
pip install --upgrade agenraci
```
